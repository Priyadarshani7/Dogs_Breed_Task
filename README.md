## Installation

1. **Git Clone**

    ```bash
   git clone https://github.com/Priyadarshani7/Dogs_Breed_Task.git
    ```

2. **Install dependencies**:

    ```bash
    npm install
    ```

3. **Start the server**

    ```bash
    npm run dev
    ```
